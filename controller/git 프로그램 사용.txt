<설치>
1. git 2.38.0을 https://git-scm.com/downloads에서 다운로드 받는다

2. 다운로드 받은 프로그램을 설치한다. 
(진행하면서 체크해야할 항목이 많은데 디폴트로 설치하시면 됩니다)

<동기화 폴더 세팅>
1. 동기화 파일 저장하려는 컴퓨터 폴더 디렉토리에서 마우스 우클릭 -> git bash 실행

2. cmd 창 뜨면 $ 표시 옆에 입력 가능 -> git clone https://github.com/hngmnj/Bbulsora.git 입력 후 엔터
(인터넷 주소는 수기로 입력해줄 것, 복사 붙여넣기 할 시 문제가 있을 수 있다고 합니다)

<동기화 폴더 pull 하기>
1. .git 폴더가 있는 디렉토리로 진입합니다.
(동기화 후 생성된 폴더 안으로 들어가시면 있습니다.)

2. 똑같이 해당 폴더에서 마우스 클릭 -> git bash 실행

3. cmd 창 뜨면 $ 표시 옆에 입력 가능 -> git pull 입력 후 엔터
(공유하려는 파일은 기존과 마찬가지로 인터넷 github에서 업로드해주시면 됩니다)

4. 동기화 실행되고 해당 디렉토리에 추가된 파일/폴더 생성

<주의사항>
로컬 저장소에서 push해서 업로드하는 기능도 있지만 원본파일이 섞이는 등 잘못될 수가 있기 때문에 우선은 업로드는

기존 방식대로 하시고, 다운로드가 필요한 경우에만 git을 활용해서 진행하는 것으로 우선 해주셨으면 합니다.
